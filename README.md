# Chaos by Ronk

אפליקציית ניהול משימות מפגישות (PWA, שחור-לבן). קובץ אחד, שמירת נתונים מקומית בדפדפן.

## הרצה מקומית
```
npm start
```
ואז לגלוש ל-http://localhost:3000

## פריסה ב-Railway
1. צור ריפו ב-GitHub ודחוף את התיקייה הזו.
2. ב-Railway: New Project → Deploy from GitHub repo → בחר את הריפו.
3. Railway מזהה Node אוטומטית, מריץ `npm start`, ומקצה כתובת ציבורית.
4. Settings → Networking → Generate Domain כדי לקבל קישור.
